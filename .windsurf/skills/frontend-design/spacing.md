# Spacing

- **Pill buttons**: `py-3 px-8` (btn-pill base), `py-2 px-5` (compact)
- **Card padding**: `p-5` ou `p-6`
- **Page container**: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- **Section gaps**: `space-y-6` ou `gap-6`
- **Card grids**: `grid gap-4 sm:grid-cols-2 lg:grid-cols-3`
- **Icon + text in buttons**: `gap-2`
- **Icon + text in nav**: `gap-2.5`
- **Label above input**: `mb-1.5`
- **Badge padding**: `px-2.5 py-0.5`
- **Nav item padding**: `px-3 py-1.5`
- **Mobile menu items**: `px-3 py-2.5`
- Content NUNCA encosta em bordas — mínimo `px-4` de padding horizontal
- Landing sections: `py-16 sm:py-24` vertical spacing between sections
